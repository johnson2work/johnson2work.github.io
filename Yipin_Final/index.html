Clark Creek Nature Area

Pick Any On The Map & Discover Nature Highlights Along 3.8 Miles of Primitive Trail

Download iNaturalist App to Help You Identify The Plants And Animals On The Trail

Rusty Shackleford at MDWFP provided FFS Consulting with the data file to create this open source web GIS project. We utilized the data from one of the state park to build a web map for the primitive trail to help promote parks and trails in Mississippi. Short-term goal is to add all trails from each state park into the web map, intermedium-term goals are to add elevation and wildlife information as layers, and finally the long-term goal is to create a user friendly web app to promote Mississippi's trails to everyone for all skill levels.
This map app provides a way for visitors to interact with several points of interest on the primitive trail in Clark Creek State Park. This project is only a prototype to showcase how easy users can access the app from a web browser in most devices from anywhere in the world in real time.
